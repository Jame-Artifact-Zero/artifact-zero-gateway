The Core Problem
Every system that handles human language accepts ambiguity as normal. NTI treats it as a structural failure.

UDDS

Upstream Denial via Downstream Substitution. The system avoids a hard constraint by over-investing in a workaround. The workaround becomes the narrative. The constraint never lands.

DCE

Deferred Constraint Externalization. The constraint is known but postponed. Planning proceeds as if unconstrained. When it surfaces, it's framed as new information.

CCA

Constraint Collapse via Aggregation. Distinct constraints get merged into a composite. Individual violations pass undetected while aggregates look compliant. The most dangerous mode — invisible by design.

When all three combine: constraints never declared, substitutes dominate, aggregates validate. Self-sealing system. No internal signal to correct.
