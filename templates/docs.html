API Documentation

Score any text for structural integrity. One endpoint. One API key. Millisecond response.

What "Score" Means

Score = V1 Only. The /score and /api/v1/score endpoints run V1 structural analysis: NII score, failure modes (UDDS/DCE/CCA), tilt patterns, dominance, and layer analysis. This is a diagnostic instrument — it tells you what's wrong. V2 (inbound stabilization) and V3 (output stabilization) are separate engines available at higher tiers. V1 scores. V2 and V3 fix.

Base URL

https://artifact0.com/api/v1

Authentication

All scoring requests require an API key passed in the X-API-Key header.

curl -X POST https://artifact0.com/api/v1/score \
  -H "Content-Type: application/json" \
  -H "X-API-Key: az_your_key_here" \
  -d '{"text": "Your text to score"}'

Quick Start

# 1. Get your API key
curl -X POST https://artifact0.com/api/v1/keys \
  -H "Content-Type: application/json" \
  -d '{"email": "you@company.com"}'

# 2. Score text
curl -X POST https://artifact0.com/api/v1/score \
  -H "Content-Type: application/json" \
  -H "X-API-Key: az_your_key_here" \
  -d '{"text": "I completely understand your concerns and everything is fine."}'

# 3. Check usage
curl -X GET https://artifact0.com/api/v1/keys/usage \
  -H "X-API-Key: az_your_key_here"
